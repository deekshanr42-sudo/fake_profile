Project Info

About This Project

🧠 Model

Random Forest Classifier with 200 trees, compared against Logistic Regression.

📊 Dataset

2,000 synthetic profiles (1,100 real + 900 fake) with realistic behavioral distributions.

🔧 Features

12 features total — 7 raw attributes + 5 engineered features capturing behavioral signals.

🎯 Results

97.5% accuracy, 97.7% precision, 96.7% recall, 0.972 F1-Score on the test set.

⚙️ Tech Stack

Python, Scikit-Learn, Flask, HTML/CSS/JS, Matplotlib, Seaborn.

📈 Future Scope

NLP analysis, network graphs, temporal patterns, deep learning.
